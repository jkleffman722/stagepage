## Priority Summary

### High Priority (affects every advancing PM, cited most often as missing)
1. **Schedule / Show Timing** — entire section missing
2. **Dock specs** (height, door dimensions, leveler) — most cited load-in gap
3. **Monitor mix count** — one of the top advance questions
4. **Console channel count + I/O format** — can't advance audio without this
5. **Rigging specs** (per-batten capacity, motor points, trim height detail) — entirely absent
6. **Phase / voltage / available amperage** — service size ≠ available power
7. **Emergency / department-specific contacts** — single contact is a failure point
8. **Packet version / date** — without this, PMs don't trust the document

### Medium Priority (important but less universally blocking)
9. **Union & Crew** — entire section missing; critical for 300+ cap venues
10. **IEM frequency band + receiver count** as structured fields
11. **Wing space** (SL / SR) as structured fields
12. **Truss positions** with load capacities (lighting section)
13. **DMX universe count + network (Art-Net / sACN)** — LD advance question
14. **Outlet types + quantities** per panel location
15. **Guest LD: can bring own console?**

### Lower Priority (nice-to-have completeness)
16. Stage surface type, deck load capacity
17. Proscenium opening dimensions
18. Conventional fixture inventory (Par 64s, Fresnels, Ellipsoidals)
19. Video: lumens, input types, media server compatibility
20. Catering details (venue-provided items per category)
21. Showers / green rooms as structured fields instead of freeform
22. Local backline rental company contact
